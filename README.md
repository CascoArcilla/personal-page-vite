# Pagina personal

Pagina web para mostrar información sobre mi como:

- Información personal
- Habilidades
- Experiencia
- Tecnologías que he usado
- Proyectos desarrollados

## Características

- Diseño responsivo
- Modo oscuro
- Animaciones simples

## Tecnologías

- HTML
- CSS
- Vanilla JavaScript

### Comandos útiles

- `npm run dev` -> Inicia el servidor en modo desarrollo
- `npm run build` -> Compila el proyecto para producción
